== Installation ==
 
Upload the files to the plugins directory and activate it. You can also install and activate directly from the admin panel.

Once activated, go to your collection and enable the view modes that you desire. You may also set it as default view mode.
 
This plugin will only work with [Tainacan plugin](https://wordpress.org/plugins/tainacan/) active, with a version greater than or equal to 0.17.3;
